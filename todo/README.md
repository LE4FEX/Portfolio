# Todo App

แอปจัดการงานที่ใช้งานง่ายพร้อมฟีเจอร์ครบครัน

## ฟีเจอร์

- ✅ เพิ่มงานใหม่
- ✅ ทำเครื่องหมายว่าทำเสร็จแล้ว
- ✅ ลบงาน
- ✅ กรองงาน (ทั้งหมด/ยังไม่เสร็จ/เสร็จแล้ว)
- ✅ ค้นหางาน
- ✅ จัดเก็บข้อมูลใน LocalStorage
- ✅ รองรับธีมมืด/สว่าง
- ✅ ออกแบบให้ใช้งานได้บนมือถือ

## เทคโนโลยีที่ใช้

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript (ES6+ Classes)

## การติดตั้งและรัน

1. Clone repository นี้
2. เปิดไฟล์ `index.html` ในเบราว์เซอร์
3. หรือรันเซิร์ฟเวอร์ local:

```bash
python3 -m http.server 8000
```

แล้วเปิด http://localhost:8000/todo/

## การใช้งาน

1. **เพิ่มงาน**: พิมพ์งานที่ต้องการทำแล้วกดปุ่ม "เพิ่ม" หรือ Enter
2. **ทำเครื่องหมายเสร็จ**: คลิกที่ checkbox ข้างงาน
3. **ลบงาน**: คลิกปุ่ม "ลบ" ข้างงาน
4. **กรองงาน**: ใช้ปุ่มกรองเพื่อแสดงเฉพาะงานที่ต้องการ
5. **ค้นหางาน**: พิมพ์คำค้นหาในช่องค้นหา

## จัดเก็บข้อมูล

ข้อมูลงานทั้งหมดจะถูกจัดเก็บใน LocalStorage ของเบราว์เซอร์ ดังนั้นข้อมูลจะยังอยู่แม้ปิดเบราว์เซอร์
